# E-Commerce Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes e-commerce sales data to identify revenue trends, customer behavior, and business performance.

## 🎯 Objective
To uncover key insights such as top customers, profitable categories, and sales trends to support business decision-making.

## 🛠 Tools Used
- SQL (PostgreSQL)
- Python (Pandas)
- Tableau

## 📊 Key Analysis Performed
- Monthly revenue trend analysis
- Top 10 customers by sales
- Category-wise profit and sales performance
- Region-wise sales distribution
- Month-over-month growth analysis

## 💡 Key Insights
- Top 15% of customers contribute nearly 60% of total revenue
- Technology category has the highest profit margins
- Sales show significant growth during Q4
- Some categories have high sales but low profitability
